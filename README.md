## Clone este projeto

```bash
> git clone https://github.com/brazzill/trump-bottt
```

## Instale as dependências:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd termux-wabot
> bash install.sh
```

### Use
```bash
> npm start
```
